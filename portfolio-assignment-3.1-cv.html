Ik heb het template ‘HTML resume’ van {pagedown} gebruikt om met R Markdown een CV als PDF bestand te genereren.

YAML header:

#---
#title: "Lijia Yu's resume"
#author: Lijia Yu
#date: "`r Sys.Date()`"
#output:
#  pagedown::html_resume:
#    self_contained: false ## set it to true for a self-contained HTML page but it'll take longer to render
#knit: pagedown::chrome_print ## this line to also produce a PDF
#---
